To run the project open the Aileron folder and guam folder in separate Matlab windows. Start the aileron simulation using the ".prj" file. In the mean time open the "Runme.m" file from the guam folder and run it. Once the Aileron simulation starts, select option 3 from the 4 available options of the Runme file. For the time being, the climb_cruise trajectory has been selected. It will run the straight flight trajectory. The current DES model runs Ok with the normal and fault conditions for this trajectory.
